## What I am Building

### IndusStream – Edge to Cloud Telemetry Platform

A cloud-native telemetry system that ingests, processes, and visualises real-time sensor data.

**Architecture:**
Arduino Sensors --> Raspberry Pi (Edge) --> AWS IoT Core --> Lambda --> DynamoDB --> S3 --> Athena --> QuickSight


**Key Features:**
- Real-time telemetry ingestion (MQTT)
- Serverless processing with AWS Lambda
- Data lake design (S3 + Athena)
- Dashboarding with QuickSight (SPICE)
- Alerting via SNS
- Partitioned analytics dataset
- Near real-time observability

[View IndusStream Project](https://github.com/ronaldmuhumuza-tech/IndusStream)
